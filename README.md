# Tender-IM-bank-skill

内蒙古地区政府招标选择商业银行项目搜索工具。面向商业银行内蒙古分行机构金融业务部，用于搜索和追踪全区14个盟市/计划单列市涉及商业银行选择的招标信息（国库现金管理、社保/医保基金存放、代理银行资格等）。

## 这是什么

这是一个 [opencode](https://opencode.ai) AI skill。它会指导 AI 自动搜索内蒙古各盟市财政厅/局、医保局官网及搜索引擎，发现银行招标机会，生成带链接的 HTML 汇总报告。

## 文件说明

```
├── SKILL.md                              # 技能定义文件（搜索规则、强制覆盖要求）
├── 内蒙古银行招标信息报告_template.html     # HTML 报告模板（含手机响应式适配）
├── 内蒙古银行招标信息报告_YYYYMMDD.html     # 生成的报告（按日期命名，git忽略）
├── LICENSE                               # MIT License
└── .gitignore                            # 忽略生成的报告文件
```

## 使用方式

在 opencode 中加载此 skill，向 AI 发出指令即可自动执行搜索和报告生成：

```
搜索内蒙古近一周的银行招标信息并生成报告
```

AI 将按照 SKILL.md 中定义的规则：
1. 全局搜索百度/必应、财政厅、阳光采购平台
2. **逐个排查全部 14 个盟市**的财政局和医保局网站
3. 汇总招标/中标信息
4. 生成手机适配的 HTML 报告

## HTML 报告特性

- **手机响应式适配**：768px / 480px 双断点，表格横向滚动，网格自动收缩
- **标题直接跳转**：每个招标/中标项目标题均为可点击链接
- **状态标签**：报名中（红）、征集中（绿）、已结束（蓝）、无结果（灰）
- **盟市覆盖网格**：14 个盟市逐项 ✅/❌ 展示

## 注意事项

- 此类招标不在公共资源交易中心（ggzyjy.nmg.gov.cn）发布
- 采招网、千里马等商业网站通常需付费查看全文
- 大部分招标发布在财政局"通知公告"栏目，搜索引擎不一定及时收录
- 该 skill 仅供内部业务参考，不构成投标决策建议
