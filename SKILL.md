# 招标选择商业银行项目搜索（内蒙古地区 + 全国级）

## 概述

招标选择商业银行的主要场景包括两类：
- **内蒙古地区**：国库现金管理、社保/医保基金定期存款、资金存放管理、代理银行资格等。这类招标**不在公共资源交易中心发布**，需要到特定渠道搜索。
- **全国级**：中央国库现金管理招标（人民银行官网发布）、全国社保基金理事会银行招选（社保基金理事会官网发布）。

### 搜索方法（每个盟市执行以下三步）

1. **财政局网站搜索**：访问 `通知公告` 或 `政府信息公开` 栏目，搜索关键词 `银行 存款 定期`、`资金存放`、`社保基金`
2. **医保局网站搜索**：访问对应栏目，搜索关键词 `银行 招标`、`竞争性存款`、`保值增值`
3. **搜索引擎补充**：用上表第三列关键词在百度/必应搜索，弥补官网未收录的情况

### 全局搜索（盟市搜索前执行）

- **百度/必应综合搜索**：`内蒙古 资金存放 银行 招标 202X`、`内蒙古 社保基金 银行 定期存款 招标`
- **公共资源交易中心**直接跳过——不收录此类招标
- **阳光采购平台**搜索"合作银行"——国资委系统项目偶尔出现
- **中国政府采购网** 搜索 `内蒙古 银行 定期存款`

### ⚠️ 阳光采购平台MCP扫描（每次任务必须执行）

阳光采购平台（nmgygcg.ejy365.com）使用 JavaScript 动态渲染，搜索引擎无法抓取项目列表，**必须使用 Chrome DevTools MCP 工具进行扫描**。

**操作步骤：**
1. 用 `chrome-devtools_new_page` 打开 `https://nmgygcg.ejy365.com`
2. 用 `chrome-devtools_fill` 在搜索框输入 `银行`
3. 用 `chrome-devtools_click` 点击"项目搜索"按钮
4. 用 `chrome-devtools_take_snapshot` 获取结果页面
5. 逐页检查 `projectType=BGGG` 的招标项目，筛选出真正的**银行招选类**项目（注意：98%以上结果是银行自身的IT采购，需仔细区分）
6. 对发现的银行招选项目，点击标题打开详情页，获取招标金额、截止日期、联系方式等完整信息
7. 如有更多关键词需要搜索（如 `维修资金`、`资金存放`、`合作银行`），重复步骤 2-5

**筛选标准：**
- ✅ 招标单位是住建局/财政局/医保局/国资委等政府机构 → 银行招选项目
- ✅ 招标内容是"银行招选"、"资金存放"、"维修资金收益银行"、"合作银行招选" → 银行招选项目
- ❌ 招标单位是"内蒙古农村商业银行"、"内蒙古银行"等银行自身 → 银行的IT/服务采购，跳过

## 生成HTML报告

搜索完成后，使用项目中的模板文件生成带链接的HTML汇总报告。

### 模板文件

`内蒙古银行招标信息报告_template.html` — 预置了完整的CSS样式（含手机响应式适配）和板块结构：

- 一、近一周招标机会（表格，含链接、金额、状态标记）
- 二、国库现金管理中标详情（排名表格，前5名）
- 三、其他历史相关项目（参考表格）
- 四、**全国级渠道招标信息（中央国库现金管理 + 全国社保基金理事会）**
- 五、**14个盟市覆盖情况（强制必填！网格化逐项展示全部14个盟市 ✅/❌）**
- 六、各渠道搜索结论（有结果/无结果渠道表格，含全国级渠道）
- 七、招标类型发布规律（频率、渠道、金额量级）
- 八、建议关注要点（卡片式布局）

### 生成方法

1. 读取模板文件获取CSS样式和结构（模板已内置手机端 @media 响应式适配）
2. 将搜索到的项目数据填入对应表格
3. 更新日期标题（文件名格式：`内蒙古银行招标信息报告_YYYYMMDD.html`）
4. **每个招标/中标项目的标题必须用 `<a href="来源URL">` 包裹**，使其可直接点击跳转到来源页面
5. **每个 `<table>` 外层必须用 `<div class="table-wrapper">` 包裹**，以支持手机端横向滚动查看表格
6. **盟市覆盖情况必须包含全部14个盟市/计划单列市，逐项标注 ✅ 已发现 / ❌ 未发现，不可缺漏**
7. **全国级渠道必须单独成章，包含中央国库现金管理招标和全国社保基金理事会招标的最新信息**
8. 来源链接列：如标题已是可点击链接，来源列可写"—"或保留补充链接（如中标公示等）
9. 状态使用 `.tag` 类标记：
   - `.tag-hot` — 红色，报名中/紧急
   - `.tag-ongoing` — 绿色，征集中
   - `.tag-done` — 蓝色，已结束
   - `.tag-none` — 灰色，无结果

### 各板块内容规则（重要）

#### 一、近两周招标机会
- **严格限定两周内**发布的项目，不得展示两周前的内容
- **优先展示内蒙古地区**的招标信息
- **中央国库现金管理**：有未开标的则展示；没有未开标的**仅展示最近一期**（即使超过两周）
- 窗口机会提示框聚焦内蒙古地区正在报名的项目

#### 三、其他相关项目
- **仅展示6个月内**（180天）的历史项目
- 超过6个月的项目不要展示

#### 四、全国级渠道招标信息
- **中央国库现金管理**：最多展示近三期，按时间倒序排列
- **全国社保基金理事会**：如本次检索未发现相关银行招标信息，则不需要展示该子板块；如仅有非招标类公告（预算、年报等）也不展示

#### 五、盟市覆盖情况
- **每次必须实现全覆盖**：14个盟市/计划单列市必须逐个搜索，不得遗漏
- 对每个盟市逐一搜索财政局和医保局官网，并补充搜索引擎搜索
- 搜索结果逐项标注 ✅ 已发现 / ❌ 未发现

### 报告关键要素

- 金额用 `<span class="amount">` 包裹，自动显示为红色加粗
- 工行中标数据用 `<strong>` 突出显示
- 对于"窗口机会"项目，使用 `<div class="highlight-box">` 高亮提醒
- 历史参考项目使用 `<div class="info-box">` 蓝色框
- 模板已内置手机版面适配（768px/480px断点），无需额外处理
- 表格在手机端自动横向滚动，城市网格自动收缩为3列/2列

## 注意事项

- 采招网、千里马等商业招标网站通常需付费/登录才能查看完整内容
- 大部分招标发布在财政厅/局官网的"通知公告"栏目，搜索引擎不一定及时收录
- 呼伦贝尔市医保局近期有自行发布招标的案例
- 住建厅、人社厅经site搜索验证不发布此类招标
- **中央国库现金管理招标**在人民银行官网发布，更新频率约为每月1-2期，金额量级远超省级
- **全国社保基金理事会**招标金额巨大（通常数百亿），为最高级别商业银行招标机会
- **阳光采购平台必须用Chrome DevTools MCP扫描**，搜索引擎无法抓取其动态渲染的项目列表。已验证该平台存在搜索引擎未收录的银行招选项目（如二连浩特维修资金银行招选）
